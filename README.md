<h3>🥇 &nbsp; Satellite Power &amp; Communication Subsystem</h3>
<h4>Space Tug in Low Earth Orbit (LEO) &nbsp;•&nbsp; Python Simulation</h4>

<p>
  <a href="https://github.com/abdullahzahid655/Satellite-Power-Communication-Subsystem-for-Space-Tug-in-LEO">
    <img src="https://img.shields.io/badge/🔗 View_Full_Project_on_GitHub-0d1117?style=for-the-badge&logo=github&logoColor=white&labelColor=0a3d6b" />

An end-to-end **aerospace engineering + Python simulation** project. I designed the complete power and communication architecture for a Space Tug meant to operate autonomously in Low Earth Orbit — combining hardware-level subsystem design with software simulation and statistical optimization.

**What makes it stand out:**
- Built a **Power Control Unit (PCU)** that manages solar energy input, battery charging cycles, and power distribution to all onboard systems
- Designed an **SDR (Software Defined Radio)** communication architecture for ground-to-satellite uplink/downlink
- Used a **Design of Experiments (DOE)** statistical pipeline in Python to optimize subsystem parameters — a technique borrowed from industrial engineering applied to space systems
- Produced full simulation output with Matplotlib visualizations of power curves, battery state, and communication windows

- Built as a reference for anyone serious about RL research or production deployment
